[적용 방법 — 1분 컷]
1) 'branding.py'를 프로젝트 루트로 복사
2) 'config.py'에 아래 한 줄 추가 (또는 기존에 있다면 생략)
   APP_URL = "https://bloodmap.streamlit.app/"
3) 'app.py'에서 st.title(...) 바로 다음 줄 근처에 다음 4줄 추가
   from config import MADE_BY
   try:
       from config import APP_URL
   except Exception:
       APP_URL = "https://bloodmap.streamlit.app/"
   from branding import render_deploy_banner
   render_deploy_banner(APP_URL, MADE_BY)

[검증 체크]
- Streamlit 실행 후 상단에 공식 배포 링크/고정 안내가 보이면 성공
- 문법 오류/띄어쓰기 문제 없음, 한국시간/면역치료 미표기 안내 포함
